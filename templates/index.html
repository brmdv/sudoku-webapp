Bram’s Sudoku Solver

This is a simple project to learn Flask, Docker, etc. The sudoku solver itself I wrote on another occasion.

Solve a sudoku

Fill in your unsolved sudoku. Use 0 for empty fields. Only the 81 digists are read, so you don't have to care about formatting.

API

Solve sudokus with /solve?s=100000006…8908000000 

  • s=
    81 digits with initial sudoku. 
  • format=
    result format, {text, html, json.
